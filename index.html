Sobre mim:

Olá,   meu nome é Valmir, sou apaixonado por tecnologia, meu primeiro contato com programação foi em 2004, quando ainda estava no ensino médio.   Apesar de ter gostado da área, só comecei a dar o devido foco em 2021, quando um amigo me apresentou o desenvolvimento web. Hoje já entendo tecnologias voltadas para o Front-end. Também gosto de “brincar” com design no figma, criando interfaces para projetos pessoais e de amigos.



Formação:

Estou cursando Ciência da Comptação na Faculdade Estácio de Sá, Nos tempos vagos gosto de desenvolver projetos envolvendo as principais tecnologias (HTML,CSS3,JAVASCRIPT).



Habilidades:

HTML: É uma linguagem de marcação utilizada na construção de páginas Web.

CSS: É um mecanismo para adicionar estilo a um documento Web.

JAVASCRIPT: É uma linguagem de programação. Juntamente com HTML e CSS, é uma das três principais tecnologias da Web.
